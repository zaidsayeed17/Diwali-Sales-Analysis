#  Diwali Sales Data Analysis using Python

##  Project Overview

This project analyzes customer purchasing behavior during the Diwali festival using a real-world sales dataset. The goal is to extract meaningful insights that can help businesses improve their marketing strategies.

## 📊 Dataset Description

The dataset contains information such as:

* Customer ID and Name
* Gender and Age Group
* Marital Status
* State and Zone
* Occupation
* Product Category
* Number of Orders
* Purchase Amount

## 🛠️ Tools & Technologies Used

* Python 
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

##  Key Insights

* Female customers contributed more to total sales compared to males
* Age group 26-35 has the highest purchasing power
* Married customers tend to spend more
* Most sales come from states like Maharashtra, Uttar Pradesh, and Karnataka
* IT, Healthcare, and Aviation professionals are major buyers

## 📈 Analysis Performed

* Data Cleaning (handling null values, removing unwanted columns)
* Exploratory Data Analysis (EDA)
* Visualization using bar charts and count plots
* Groupby analysis for deeper insights

##  Conclusion

The analysis shows clear trends in customer behavior during festive seasons. Businesses can target specific demographics (like females aged 26–35) and high-performing states to increase sales.
